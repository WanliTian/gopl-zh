### 从源文件构建:

1. 运行 `go get github.com/golang-china/gopl-zh`, 获取 源文件
2. 运行 `npm install gitbook-cli -g`, 安装 GitBook 命令行工具
3. 运行 `go generate github.com/golang-china/gopl-zh`, 生成 `_book` 目录
4. 打开 `_book/index.html` 文件
